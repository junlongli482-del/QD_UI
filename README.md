# QD_UI

教学平台 UI 重设计与移动端适配项目。

本仓库用于沉淀产品侧交付材料，包括原平台模块梳理、第一阶段原版 Demo 还原范围、移动端兼容性问题清单、UI 改版说明和后续给开发团队的验收标准。

## 当前阶段

第一阶段：原版还原 Demo 与移动端问题梳理。

目标不是直接开发生产代码，而是先基于现有平台做一个可展示的纯前端原型，确保页面结构、功能模块和关键流程没有遗漏，再进入 UI 改版和开发交付说明。

## 已确认原则

- 先还原原平台，再讨论改版。
- 先解决手机端可用性和显示问题，同时覆盖网页端 UI。
- 不接真实后端接口，不提交真实业务数据。
- Demo 用于领导评审和后续开发沟通，不作为生产系统。
- 产品文档只记录已确认信息，推测和建议单独标注。
